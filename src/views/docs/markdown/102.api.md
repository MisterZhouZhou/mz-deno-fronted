接口描述： 图形验证码验证接口

请求方式：`POST`

请求示例： `http://localhost:8000/api/verify`

<br/>

**请求参数说明**：
| 参数名称 | 是否必填 | 类型 | 说明 |
| -------- | -------- | -------- |-------- |
| code | 是 | string|number | 图形验证码 |

<br/>

**返回参数说明**：

| 参数名称  | 类型 | 说明 |
| -------- | -------- |-------- |
| _ | _ | 返回的文件流 |

<br/>

**返回示例**
```js
{
  "code": 400,
  "data": null,
  "msg": "验证码位数错误"
}
```
