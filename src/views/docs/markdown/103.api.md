接口描述：生成不同尺寸的占位图

请求方式：`POST`

请求示例： `http://localhost:8000/api/placeholder/100x100`
> 其他产品如：https://via.placeholder.com/50x50

<br/>

**请求参数说明**：
| 参数名称 | 是否必填 | 类型 | 说明 |
| -------- | -------- | -------- |-------- |
| size | 是 | string|number | 占位图尺寸 |

<br/>

**返回参数说明**：

| 参数名称  | 类型 | 说明 |
| -------- | -------- |-------- |
| _ | _ | 返回的文件流 |

<br/>

**返回示例**
```js
{
  "code": 400,
  "data": null,
  "msg": "验证码位数错误"
}
```
