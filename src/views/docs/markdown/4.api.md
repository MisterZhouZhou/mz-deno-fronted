接口描述：获取文件预览接口

请求方式：`GET`

请求示例： `http://localhost:8000/uploads/xxx.png`

<br/>

**请求参数说明**：

| 参数名称 | 是否必填 | 类型 | 说明 |
| -------- | -------- | -------- |-------- |
| :file | 是 | string | 上传的文件名称 |

<br/>

**返回参数说明**：

| 参数名称  | 类型 | 说明 |
| -------- | -------- |-------- |
| _ | _ | 返回的文件流 |

<br/>

**返回示例**
无
