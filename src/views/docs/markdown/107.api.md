接口描述：根据内容生成二维码接口，返回html

请求方式：`GET`

请求示例： `http://localhost:8000/api/qrcode/show?data=nihao&size=300`

<br/>

**请求参数说明**：
| 参数名称 | 是否必填 | 类型 | 说明 |
| -------- | -------- | -------- |-------- |
| data | 是 | string | url或者文本 |
| size | 否 | string|number | 生成的二维码尺寸，默认200 |

<br/>

**返回参数说明**：

| 参数名称  | 类型 | 说明 |
| -------- | -------- |-------- |
| _ | _ | 返回的html |

<br/>

**返回示例**
无
